# 记录每一个实验对象的变异体被杀死的信息

## 利用RT筛选变异体
随机生成10000个测试用例，然后在每一个变异体上
执行10000个测试用例，判断他们的结果与源程序是否相等，最后统计
每个变异体可以被多少测试用例杀死。

### ACMS程序变异体的信息

失效率在0.1以下0.05以上的故障有：SDL_10, SDL_8
